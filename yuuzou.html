WebーDesign

HTML CSS JavaScript(jQuery)を使って作ったWebサイトです。
以前、学習する際に作成したWebサイトをベースに、「自分が今まで描いたイラストが商品になったら」をテーマに架空の雑貨店のHP風に仕上げています。
CSSのバックグラウンドイメージでヘッダーやフッターにライン状のイラストを入れています。
イラストの画像をクリックすると商品画像が背景を薄暗くして表示する、という動きを一から考えてコーディングしました。
こちらもAdobe Illustratorを使用する前の作品なのでデザインは全て別の描画ツールを使用しています。
三つ目に作成したWebサイトです。
