HTML CSS JavaScript(jQuery)を使って作ったWebサイトです。
初めてのポートフォリオということで、他の方々のものを参考にしながら自分のポートフォリオを作成しました。
Webサイトだけの紹介では、と過去にデザインしたイラストも紹介しています。
ライブラリを使い、画像をクリックすると全体図が見える作りにしてる他、CSSで使い方を調べ初めてトリミングをして画像を配置をしました。
こちらもAdobe Illustratorを使用する前の作品なのでデザインは全て別の描画ツールを使用しています。
今回のコンタクトフォームはjQueryを用いて必須項目が未記入の場合エラー表示を出しページ遷移しない、というのを一から実装しています。
四つ目に作成したWebサイトです。
